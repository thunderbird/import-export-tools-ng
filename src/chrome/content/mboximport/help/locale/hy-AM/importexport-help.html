ImportExportTools NG- ը հետագա ընդլայնում է, որը հիմնված է Պաոլո «Կաոսմոսի» բնօրինակ աշխատանքի վրա: Ընդլայնումն առաջարկում է բազմաթիվ ներմուծման և արտահանման գործառույթներ, որոնք աջակցում են մի քանի տարբեր ֆայլերի ձևաչափերի, ներառյալ ՝ Նշում. Որոշ ձևաչափեր չեն աջակցվում բոլոր գործառույթներում:

- EML
- maildir
- HTML
- պարզ ենթատեքստ

- PDF

Ամսաթվի հատուկ ձևաչափն օգտագործում է% հաջորդականություն, որը համապատասխանում է strftime() Unix- ի նման գործառույթին: Այն հիմնված է strftime() JavaScript նավահանգստի վրա, որը ֆունկցիա է ձևաչափելու ամսաթիվը և ժամանակը:
Նավահանգիստը () TH Doan- ի կողմից (https://thdoan.github.io/strftime/)

Հաջորդականություն Նկարագրություն
%a Շաբաթվա օրվա կրճատ անվանումը.
%A Շաբաթվա օրվա լրիվ անվանումը.
%b Ամսագրի կրճատ անվանումը
%B Ամբողջ ամսվա անվանումը:
%c Ընթացիկ տեղանքի համար նախընտրելի ամսաթիվը և ժամանակը (UTC) ներկայացում:
%C Դարի համարը (տարի / 100) որպես երկնիշ ամբողջ թիվ:
%d Ամսվա օրը որպես տասնորդական թիվ (01-ից 31-ի սահմաններ):
%e Ամսվա օրը որպես տասնորդական թիվ (1-ից 31-ի սահմաններ):
%F ISO 8601 ամսաթվի ձևաչափ (համարժեք %Y-%m-%d ):
%G ISO 8601 շաբաթվա վրա հիմնված տարին `դարը որպես տասնորդական թիվ: 4 նիշանոց տարին համապատասխանում է ISO շաբաթվա համարին (տես %V ): Սա ունի նույն ձևաչափը և արժեքը, ինչ %Y , բացառությամբ, որ եթե ISO շաբաթվա համարը պատկանում է նախորդ կամ հաջորդ տարվան, դրա փոխարեն օգտագործվում է այդ տարին:
%g %G նման, բայց առանց դարի, այսինքն ՝ երկնիշ տարի (00-99):
%H Asամը որպես տասնորդական թիվ 24 ժամյա ժամացույցի միջոցով (միջակայքը 00-ից 23-ը): Տե՛ս նաև %k :
%I Asամը որպես տասնորդական թիվ ՝ օգտագործելով 12-ժամյա ժամացույց (01-ից 12-ի սահմանաչափ): Տես նաև %l :
%j Տարվա օրը որպես տասնորդական թիվ (միջակայքը 001-ից 366):
%k Aամը որպես տասնորդական թիվ ՝ օգտագործելով 24-ժամյա ժամացույց (միջակայք ՝ 0-ից 23): Տե՛ս նաև %H :
%l Asամը որպես տասնորդական թիվ ՝ օգտագործելով 12-ժամյա ժամացույց (միջակայքը 1-ից 12-ը): Տես նաև %I :
%m Ամիսը որպես տասնորդական թիվ (միջակայքը 01-ից 12-ը):
%n Ամիսը որպես տասնորդական թիվ (1-ից 12-ի սահմաններ):
%M Uteամկետը որպես տասնորդական թիվ (միջակայքը 00-ից 59):
%p Կամ «AM» կամ «PM» ըստ տրված ժամանակի արժեքի: Կեսօրին վերաբերվում են որպես «PM», իսկ կեսգիշերին `որպես« AM »:
%P %p նման, բայց փոքրատառերով («am» կամ «pm»):
%s Վայրկյանների քանակը դարաշրջանից ի վեր, 1970-01-01 00:00:00 +0000 (UTC):
%S Երկրորդը ՝ որպես տասնորդական թիվ (միջակայքը 00-ից 59):
%u Շաբաթվա օրը որպես տասնորդական (1-ից 7-ի միջակայք), երկուշաբթի օրը `1. Տե՛ս նաև %w :
%V Ընթացիկ տարվա ISO 8601 շաբաթվա համարը որպես տասնորդական թիվ (միջակայք 01-ից 53), որտեղ շաբաթ 1-ը առաջին շաբաթն է, որն ունի նոր տարվա առնվազն 4 օր (այսինքն `առաջին հինգշաբթի):
%w Շաբաթվա օրը որպես տասնորդական (տող 0-ից 6), իսկ կիրակի օրը `0. Տե՛ս նաև %u :
%x Առանց ժամանակի ընթացիկ տեղանքի համար նախընտրելի ամսաթվի ներկայացում:
%X Ընթացիկ տեղանքի համար նախընտրելի ժամանակի ներկայացում առանց ամսաթվի:
%y Տարին որպես տասնորդական թիվ առանց դարի (միջակայքը 00-ից 99):
%Y Տարին որպես տասնորդական թիվ ներառյալ դարը:
%z +hhmm կամ -hhmm թվային ժամային գոտին (այսինքն ՝ UTC- ից ժամ և րոպե օֆսեթ):
%Z Zամանակային գոտու անվանումը կամ հապավումը:

Համատեղելիության նշումներ.

  • %c - ֆորմատավորված տողը մի փոքր այլ է
  • %D - չի իրականացվել (օգտագործել %m/%d/%y կամ %d/%m/%y )
  • %e - տարածք չի ավելացվում
  • %E - չի իրականացվել
  • %h - չի իրականացվել (օգտագործել %b )
  • %k - տարածք չի ավելացվում
  • %n - նման է %m , բայց չկա առաջատար զրո (օգտագործեք \n նոր գծի համար)
  • %O - չի իրականացվել
  • %r - չի իրականացվել (օգտագործեք %I:%M:%S %p )
    • ${subject} Ուղերձի թեման
  • %R - չի իրականացվել (օգտագործեք %H:%M )
  • %t - չի իրականացվել (օգտագործել \t )
  • %T - չի իրականացվել (օգտագործեք %H:%M:%S )
  • %U - չի իրականացվել
  • %W - չի իրականացվել
  • %+ - չի իրականացվել
  • %% - չի իրականացվել (օգտագործել % )

Ընդլայնված ֆայլի անվան ձևաչափումը օգտագործում է նշանները: Ֆայլի ճշգրիտ անունը կարելի է կառուցել ՝ օգտագործելով նշանները, ինչպես նաև ցանկացած տեքստային կամ վավեր ֆայլի անվան նիշերը: Անգլերեն (en-US) նշանները հասանելի են ցանկացած տեղական տարածքում: Տեղական հատուկ նշանները նույնպես մատչելի են և նշված են անգլերեն աղյուսակի տակ:

Օրինակներ.

${subject}-${date}-${index} - Լռելյայն ձևաչափ
${prefix}_${date_custom}_${sender} - Լռելյայնի փոխարեն օգտագործել ամսաթվի ձևաչափը

Կան մի քանի նոր առանձնահատկություններ և ընտրանքներ, որոնք ընտրանքների երկխոսության մեջ դեռ կարգաբերելի չեն: Այս հատկությունները կայունացնելուց հետո դրանք կավելացվեն UI- ին:
Դրանց օգտագործման համար դրանք պետք է դրվեն Config Editor- ի մեջ: Խմբագիր մուտք գործելու համար օգտագործեք Գործիքներ / Ընտրանքներ ընտրացանկը, ապա ընտրեք Ընդլայնված բաժինը և ներքևում կլինի «Կազմաձևման խմբագիր» կոճակը: Եթե որոնման տողում մուտքագրեք «importexporttoolsng», կստանաք ընդլայնման բոլոր նախապատվությունները: Դրանից հետո, ըստ անհրաժեշտության, կարող եք խմբագրել ստորև ներկայացված տարբեր նախասիրությունները:

importexporttoolsng.export.index_date_custom_format

Այս նախապատվությունը հետևում է նույն ամբողջական ձևաչափմանը, որն օգտագործվում է ինդեքսային ֆայլերի համար: Դա լռելյայն դատարկ տող է, եթե ֆորմատավորված ամսաթիվ / ժամանակի տող ունի, ապա դա կփոխի ինդեքսի ամսաթվի ձևաչափը:

importexporttoolsng.export.attachments.filename_extended_format

importexporttoolsng.export.embedded_attachments.filename_extended_format

Այս երկու նախապատվություններն էլ օգտագործում են նույն տեսակի ընդլայնված ֆայլերի անվանման ձևաչափեր, որոնք օգտագործում է արտահանված հաղորդագրությունների ֆայլի անունը: Առաջինը `արտաքին կցորդների համար, երկրորդը` ներկառուցված (inline) կցորդների:

importexporttoolsng.experimental.use_delivery_date

Հիմնական, գլոբալ Hotkey (դյուրանցումներ) առանձնահատկություն

importexporttoolsng.experimental.hot_keys

Այս նոր նախապատվությունը օգտվողին թույլ է տալիս կարգավորել մինչև ստեղնաշարի 10 դյուրանցում: Դյուրանցումները կարող են կատարել ImportExportTools NG գործառույթների մեծ մասը, ինչպես նաև Thunderbird- ի այլ գործառույթներ: Նախապատվության տողը ձևաչափվում է որպես JSON ֆորմատավորված օբյեկտ: Ձևաչափը և ստեղները հետևյալն են.

Տաք ստեղների գրառումների օրինակ (JSON օբյեկտների զանգված)

id - հիմնական ցուցիչ (1-10)
key - բանալին (մեկ նիշ)
keycode - «VK_ *» ստանդարտ ծածկագիրը
Կարող է ներառվել կամ բանալին, կամ բանալիը, բայց ոչ երկուսը
modifiers - հիմնական փոփոխիչները, տարածությունից առանձնացված ցուցակը («հսկողություն», «հերթափոխ», «alt», «accel»)
oncommand - ուղղակի հրամանի տող
contexts - պատուհանի համատեքստ, որտեղ բանալին ակտիվ է
'all' - բոլոր պատուհանները մեսենջերի տակ, կազմել, որոնել
'messenger' - մեսենջերի հիմնական պատուհանը և դրա հետնորդները
'compose' - հաղորդագրությունը կազմելու պատուհան
'search' - որոնման երկխոսություն
